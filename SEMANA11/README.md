Práctica de pruebas unitarias en PHP utilizando PHPUnit para validar excepciones mediante `expectException` y `expectExceptionMessage`.

Requisitos

PHP 8.2 o superior
Composer
PHPUnit 11

Instalación

Clonar el repositorio:

git clone URL_DEL_REPOSITORIO

Ingresar al proyecto:

cd Practica-S11

Instalar dependencias:

composer install


Generar el autoload:

composer dump-autoload

Funcionalidades

Calculadora
División
Raíz cuadrada
Factorial

Validador

Validación de edad
Validación de correo electrónico
Validación de contraseña

Ejecutar pruebas

php vendor/bin/phpunit

Ejecutar pruebas con TestDox

php vendor/bin/phpunit tests --testdox

Resultado esperado

OK (17 tests, 24 assertions)
